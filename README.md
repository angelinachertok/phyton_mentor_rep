Назначение
Этот репозиторий содержит:

Эталонные решения для проверки (good/perfect)
Тестовый раннер run.py для автоматической проверки
Как работает
Студенты форкают этот репозиторий
Добавляют свои решения в файлы lesson_X.py
SandboxApi запускает run.py для проверки решений
Результаты сохраняются в results.json
Тестирование
Запуск тестов:

python homework/run.py
Результаты будут в homework/TestResults/results.json
